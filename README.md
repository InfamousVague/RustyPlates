# RustyPlates
Weightlifting Guide &amp; Logbook.

This is a playground for me to practice Rust and not intended to be a complete "product".

Goals:

Add and modify workouts.
Add and modify equiptment and allow adding "required equiptment" to the workouts.
Add and modify sets and reps.
Automatically adjust weight up as the plan continues.
Allow adding and modifying custom weights.
Create weekly schedules for the workouts and set frequency.
